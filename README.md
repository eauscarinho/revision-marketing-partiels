# Marketing Arena

Plateforme de revision pour les partiels de marketing operationnel, construite a partir du PDF de cours.

## Fonctionnalites

- Banque de 104 questions couvrant l'ensemble des notions du cours.
- Quiz sans reponse ouverte: QCM, vrai/faux et choix multiples.
- Score, suite de bonnes reponses, precision et meilleur score sauvegarde en local.
- Page de suivi persistante avec historique, progression par theme et recommandations.
- Indices qui coutent des points.
- Modes cours complet, entrainement par theme, choix multiples et session courte.
- Fiches rapides par theme.
- App statique compatible GitHub Pages.

## Lancer en local

Ouvrir `index.html` dans un navigateur, ou lancer un serveur statique:

```powershell
python -m http.server 4173
```

## Hebergement gratuit

Le projet est pret pour GitHub Pages: publier la branche principale, puis activer Pages avec la source `Deploy from a branch`, dossier `/root`.
